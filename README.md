HtmlValidatorWithSelenium
=========================

Using Selenium to grab page contents, and then calling validator.w3.org to validate the page.

Build
=====

* Maven 3.1.1
* jdk1.8.0_20.jdk
* PhantomJS 1.9.7

```
mvn package
```

Run
===

```
java -jar target/HtmlValidatorWithSelenium-1.0.0-SNAPSHOT.jar
```
